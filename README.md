# 🤖 HH Job Search Bot (aiogram)

Telegram-бот для поиска вакансий через API hh.ru, написанный на aiogram.
Позволяет быстро настраивать фильтры и получать свежие вакансии прямо в Telegram.

🚀 Возможности

🌍 Фильтр по странам

Россия

Казахстан

Беларусь

🏙 Выбор крупных городов внутри выбранной страны

⚙️ Удобная настройка фильтров

После каждого выбранного фильтра бот предлагает:

⚙ Настроить ещё
🔎 Начать поиск

Это позволяет быстро изменить параметры перед запуском поиска.

🔄 Обновление вакансий

Бот запоминает последний запрос пользователя.

Кнопка:

🔄 Обновить вакансии

позволяет получить новые результаты по тем же фильтрам.

💾 Сохранение вакансий

Все найденные вакансии сохраняются в:

SQLite

Это позволяет:

избегать повторов

хранить историю

анализировать вакансии

🛡 Защита от спама

Чтобы избежать перегрузки API:

реализован cooldown между запросами

пользователь не может отправлять запросы слишком часто

📌 Как работает

Пользователь запускает бота

Выбирает страну и город

Настраивает фильтры (зарплата, формат работы)

Нажимает «начать поиск»

Бот отправляет список вакансий

Можно нажать «обновить», чтобы получить новые вакансии по тому же запросу

📦 Установка
git clone https://github.com/yourusername/hh-job-bot
cd hh-job-bot

pip install -r requirements.txt
Бот отправит список вакансий и предложит:

🔄 Обновить вакансии
